# UD4 – Arrays y Colecciones

## Descripción

Ejercicios centrados en el uso de arrays y colecciones en Java.  
Se trabajan la manipulación de datos, recorridos y operaciones básicas sobre estructuras de datos.

## Contenido

| Ejercicio | Descripción |
|-----------|-------------|
| EjercicioArray.java | Manejo de arrays, recorrido e impresión de elementos. |
| ListaAlumnos.java | Uso de ArrayList para almacenar objetos Alumno. |

## Cómo ejecutar

1. Abrir Eclipse IDE.  
2. Copiar los archivos `.java` al proyecto.  
3. Ejecutar cada archivo y verificar los resultados en la consola.

## Conceptos aplicados

- Arrays y ArrayList  
- Bucles `for` y `for-each`  
- Manipulación de colecciones
