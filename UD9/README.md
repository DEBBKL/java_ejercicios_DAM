# UD9 – Proyectos y Repaso General

## Descripción

Ejercicios de repaso y proyectos integradores de Java, donde se aplican conceptos de POO, colecciones, archivos y excepciones.

## Contenido

| Ejercicio | Descripción |
|-----------|-------------|
| Proyecto1.java | Proyecto integrador aplicando POO y manejo de datos. |
| Proyecto2.java | Proyecto de repaso que incluye arrays, listas y ficheros. |

## Cómo ejecutar

1. Abrir Eclipse IDE.  
2. Copiar los archivos `.java` al proyecto.  
3. Ejecutar cada proyecto y revisar los resultados en la consola.

## Conceptos aplicados

- Integración de POO, arrays, colecciones y manejo de archivos  
- Buenas prácticas de Java  
- Resolución de problemas complejos
