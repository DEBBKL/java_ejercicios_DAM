# UD6 – Archivos y Entrada/Salida

## Descripción

Ejercicios centrados en la manipulación de archivos, lectura/escritura de datos y flujos de entrada/salida en Java.

## Contenido

| Ejercicio | Descripción |
|-----------|-------------|
| LeerArchivo.java | Lee datos desde un archivo de texto y los muestra en consola. |
| EscribirArchivo.java | Escribe datos en un archivo de texto desde Java. |

## Cómo ejecutar

1. Abrir Eclipse IDE.  
2. Copiar los archivos `.java` al proyecto.  
3. Ejecutar cada archivo y verificar que los datos se leen/escriben correctamente.

## Conceptos aplicados

- Manejo de archivos con FileReader/FileWriter  
- Streams de entrada y salida  
- Manejo de excepciones de I/O
