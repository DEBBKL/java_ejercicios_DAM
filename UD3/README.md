# UD3 – Programación Orientada a Objetos

## Descripción

Esta carpeta contiene los ejercicios de la Unidad Didáctica 3, centrados en la Programación Orientada a Objetos (POO) en Java.  
Se trabajan clases, objetos, métodos, encapsulación y constructores.

## Contenido

| Ejercicio | Descripción |
|-----------|-------------|
| Vehiculo.java | Crea una clase Vehículo con atributos y métodos para acelerar y frenar. |
| Circulo.java | Implementa una clase Círculo con cálculo de área y sobrecarga de métodos. |
| Texto.java | Clase que maneja texto y cuenta palabras. |

## Cómo ejecutar

1. Abrir Eclipse IDE.  
2. Copiar los archivos `.java` al proyecto.  
3. Ejecutar cada clase y observar los resultados.  

## Conceptos aplicados

- Clases y objetos  
- Métodos y sobrecarga  
- Encapsulación y atributos privados  
- Constructores
