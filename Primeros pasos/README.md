# Primeros Pasos

## Descripción

Esta carpeta contiene los ejercicios iniciales de Java, diseñados para familiarizarse con la sintaxis básica, entrada/salida, operadores y estructuras de control.

## Contenido

| Ejercicio | Descripción |
|-----------|-------------|
| Ejercicio1.java | Imprime mensajes por consola y maneja variables simples. |
| Ejercicio2.java | Realiza operaciones matemáticas básicas y muestra resultados. |
| Ejercicio3.java | Uso de condicionales `if` y `switch`. |
| Ejercicio4.java | Bucles `for` y `while` aplicados a problemas sencillos. |

## Cómo ejecutar

1. Abrir Eclipse IDE.  
2. Importar esta carpeta como proyecto existente.  
3. Ejecutar cada archivo `.java` desde Eclipse.  
4. Revisar los resultados en la consola.

## Conceptos aplicados

- Sintaxis básica de Java  
- Tipos de datos y variables  
- Operadores aritméticos y lógicos  
- Estructuras condicionales y bucles
