# UD5 – Programación Funcional y Excepciones

## Descripción

Ejercicios que incluyen el manejo de excepciones, estructuras de control avanzadas y programación funcional básica en Java.

## Contenido

| Ejercicio | Descripción |
|-----------|-------------|
| Excepciones.java | Manejo de excepciones con try-catch-finally y throws. |
| Streams.java | Uso de Streams para operaciones funcionales sobre colecciones. |

## Cómo ejecutar

1. Abrir Eclipse IDE.  
2. Copiar los archivos `.java` al proyecto.  
3. Ejecutar cada archivo y revisar resultados en la consola.

## Conceptos aplicados

- Manejo de excepciones  
- Programación funcional básica  
- Streams y operaciones sobre colecciones
